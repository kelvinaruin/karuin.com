Hybrid OU Structure

The following is a Hybrid environment OU structure meant to guide and standardize all clientsâ€™ Active Directory OU structure to support Entra ID Connect and/or Entra ID Cloud Sync agents.

This structure will also work for a non-hybrid environment while preparing the OU structure to be hybrid.

By incorporating this OU structure, we can standardize the majority of onboarding and offboarding user accounts for hybrid environments. We can also have every environment ready to be upgraded to Entra ID services.

Naming Standards

Security Group Naming Standards:

The following security group OUs will require a naming standard to easily identify what their intention is for use within the hybrid environment.

Global Security Groups

A global security group is any group that is managed in Active Directory for membership and used to assign permission for access to Entra ID services or functions.

Naming standard

XXX-G-ServiceName

XXX = the three-letter custom company acronym

G = Global

ServiceName

Any service that is hosted in Entra ID and requires association for permission to use this service.

The same naming standard is applied to the third-level OUs as well under the Global Security OU

Azure

M365 Licensing

Add a description identifying the service that is being applied in Entra ID using the group

Application Security Groups

An application security group is any line of business application that is hosted on-premises and requires membership to access the LoB application.

Naming standard

XXX-APP-ApplicationName

XXX = the three-letter custom company acronym

APP = Application

ApplicationName

Add a description identifying the application and server resources using this group

Folder Security Groups

A folder security group is any folder within any file share that requires NTFS security permissions for accessing the files beneath it.

Naming standard

XXX-FS-FolderName

XXX = the three-letter custom company acronym

FS = File Security

FolderName

Add a description and include the path to the folder this group is applied to

Printer Access Security Groups

A printer security group is used for any printer that requires NTFS permissions to enable access for the privilege to use the printer.

Naming standard

XXX-MFP-PrinterName

XXX = the three-letter custom company acronym

MFP = Mulit-Function Printer

PrinterName

Add a description and include the printerâ€™s name and port this group is applied to
