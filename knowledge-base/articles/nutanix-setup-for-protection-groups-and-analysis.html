Summary

Protection Groups are configured on Nutanix to create local snapshots of the servers themselves. These are not long term snapshots and do not replace having backups to a device like a Datto, but these are nice and handy to have.

Create the Protection Groups

Log into Prism Element

From the drop down select Data Protection

Click + Protection Domain in the top right (1)

Select Async DR (2)

Enter a name and click Create

Update Protection Groups (add/remove servers, modify schedules)

Log into Prism Element

From the drop down select Data Protection

Right Click on RKE-Server-Snapshot

Select Update

Select the server from the Unprotected Entities on the left

Click Protect Selected Entities at the bottom

The new entity will show up on the right. Click Next

Click Close

Restore from Snapshot

Log into Prism Element

From the drop down select Data Protection

To the right of the snapshot you would like to restore from click Restore

Choose the method for the Restore (Overwrite Existing entity or Create a new one), select the server, and click OK

Create Analysis charts

Log into Prism Element

From the drop down select Analysis

Create a new Metric chart for the Metrics Memory Usage, Network Rx, and Network Tx. Memory Usage will have all VMs. The Network Rx and Network Tx will also include the CVMs
