Introduction

When migrating a user from an AD domain profile to an Entra ID profile normally, all data will be lost and the system will act as if the new profile is an entirely new one, since the system will generate a new profile folder.

A good way to resolve this issue is to use a tool called Profile WizardÂ to modify the new profile so it points to the old profile's folder. This allows the following things to carry over seamlessly from the previously-used profile:

Files

Desktop/Taskbar/Start Menu/System TrayÂ Shortcuts and settings

Certain application settings

Usage of this tool can ensureÂ a smooth transition from Active Directory to Entra ID.

Prerequisites and Additional Information

To start, you must have a PC that's in the following state:

Not joined to an Active Directory Domain

Joined to Entra ID but prior to the first login of the account that joined Entra ID

Logged into a local administrator account (NOT a domain or Entra ID account)
