User VMWare Converter to migrate a server (physical or virtual) to vCenter or vSphere

Utilize INC / Take control and log on to the machine that is to be migrated.

Download and install the VMWare Converter 6.2.0 executable.

Use the default options during the installation, such as local installation as shown below:

Restart the server.

Turn off any services that may interfere with the migration (SQL, etc.)

Turn off Windows firewall, User Access Control (UAC), and Anti-Virus, if possible.

Create a maintenance window for this server in INC.

Make sure you have a good and current backup of your server.

Create a checkpoint or snapshot of your server, if possible.

Log into the Datto Portal and pause Datto backups (cloud and local) for this server.

Send an email to the client/parties involved informing them of the migration start.

Launch the VMWare converter.

Once the machine is done converting, before starting on the Vcenter side make sure you disable the NIC on the source server.

Copy the source server MAC address and assign it to the NIC on the destination server.
