Backup and Restore ESXI Configurations

The destination build number of the host matches the build from which the backup was taken.

The UUID of the host remains the same.

p the ESXi host configuration data

To synchronize the configuration changed with persistent storage: run this command:

vim-cmd hostsvc/firmware/sync_config

To backup the configuration data for the ESXi host, run this command:

vim-cmd hostsvc/firmware/backup_config

The IP address of your ESXi host can be checked in the ESXi direct console interface or using the following command:
